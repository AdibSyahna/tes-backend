# Tes Backend
## Author
[Adib Syahna](https://github.com/AdibSyahna)
## Environment Requirements
- Node.JS v24.11.1 or higher
## Installation
1. Clone this repository:
    ```bash
    git clone https://github.com/AdibSyahna/tes-backend.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd tes-backend
    ```
3.  Rename `.env.example` to `.env`
4.  Edit `.env` file and set your database connection details
5. Open `run.bat` OR run `npm i && npm start` in terminal
